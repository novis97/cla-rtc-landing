# Landing Page — Rumah Tahfizh Center

Landing page Gerakan Sedekah Subuh Sahabat Tahfizh. HTML statis, siap deploy ke Vercel.

## Struktur
- `index.html` — halaman utama
- `images/` — logo + 9 foto dokumentasi
- `robots.txt`, `llms.txt` — biar mudah ditemukan Google & AI
- `.gitignore` — biar file sampah tidak ikut ter-upload

## Cara update (via GitHub Desktop)
1. Edit file di folder ini (teks/foto)
2. Buka GitHub Desktop → tulis ringkasan perubahan → **Commit to main**
3. Klik **Push origin**
4. Vercel deploy otomatis dalam ~30 detik

## Deploy pertama kali (Vercel)
1. vercel.com → Add New → Project → Import repo ini
2. Framework Preset: **Other**
3. Deploy
